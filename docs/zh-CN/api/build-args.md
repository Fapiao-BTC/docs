# 构建参数

## 接口概述

该接口提供将合约交互参数,转为Hex返回

## 请求参数

```json
{
	"kid": "",
	"method": "",
	"args": []
}
```

## 请求示例

### 请求URL

`http://127.0.0.1:8765/api/build/arg`

### 请求方法

POST

### 请求示例参数

```json
{
	"kid": "kfce4982e1f4703ba123238d9d28dcf70f7e97df49efeef7086392655512e",
	"method": "sendMethod",
	"args": [
		"12345"
	]
}
```
