# 快速创建Inscribe

**我们在协议中内置了一个Inscribe模板。您可以通过继承INSC类快速创建铭文**


**还可以通过重写INSC函数来完成所需的业务逻辑**

```javascript
class Contract extends INSC{

     init() {
         this._name = 'KFC-INSC';
         this._symbol = 'KI';
         this._mintMax = SafeMath.val(1000);
         this._totalSupply = SafeMath.val(0);
         this._totalSupplyMax = SafeMath.val(1000000);
     }

}
```
