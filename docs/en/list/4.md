# Call between contracts

```javascript
class Contract{

     init(){

     }

     //Call between contracts
     testCall(){
         //Create contract instance
         let k = new Caller("kid");
         //Call the contract read-only function
         k.call("sayHello");
         //Call the contract writing function
         k.call("setHello","World");
     };

}
```
